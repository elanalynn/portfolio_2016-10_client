{{vm.projectList[0].name}} is a storytelling platform.

As a user, you can create an account by signing up with Twitter. You can browse stories, add chapters to any other user stories and create your own stories. You can choose a genre or leave it open and update story details at any point. As a user, you also have a dashboard where you can track the stories you own, the stories you've contributed to and the stories you've marked as favorites.

Upcoming features include the ability to approve or reject contributions, track a story's progress across the world with an interactive map and drawing comic book style panels for chapter contributions.

Features include:

  • Login with Twitter
  • Create stories and edit details at any time
  • Contribute to any story by writing new chapters
  • Search for stories by genre
  • Save favorites
  • Keep track of all activity via Dashboard
